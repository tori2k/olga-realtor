# olga-realtor

Сайт для Ольги Левчук — эксперта по недвижимости в Москве.
Astro 6 + Tailwind 4. Содержимое страницы — в `src/content.ts`.

## Запуск

```
pnpm install   # или npm install
pnpm dev       # http://localhost:4321
pnpm build
```

## Что нужно от Ольги перед публикацией

- Профессиональное фото (горизонтальное и вертикальное, формат 4:5)
- Реальные отзывы клиентов (3+)
- Реальные кейсы со сделками: что было, что стало, фото объекта (можно без адреса)
- Подтверждение цифр в блоке «Обо мне» (стат. блок: лет опыта, число клиентов)
- Решение по доменy (предполагаю `olga-levchuk.ru`)
